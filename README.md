# 🐾 AI-Powered Pet Care Assistant

A modern full-stack AI-powered pet care platform built using React, Tailwind CSS, Flask, and Groq LLM APIs.

This project helps pet owners get intelligent guidance related to pet health, nutrition, behavior, and wellness through an interactive AI chatbot interface.

---

## ✨ Features

* 🤖 AI-powered pet care chatbot
* 🩺 Intelligent symptom guidance
* 🎨 Premium modern UI inspired by Heads Up For Tails
* ⚡ Fast React + Tailwind frontend
* 🐍 Flask backend API
* 🔗 Frontend-backend REST API integration
* 🔐 Secure environment variable handling
* 📱 Fully responsive design
* 🧠 Prompt-engineered AI responses

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/KartikeyGogia/Pet-care-ai-assistant.git
```

---

## Frontend Setup

```bash
cd petcare-ui
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```

---

## Backend Setup

```bash
cd petcareaisystem
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

Backend runs on:

```bash
http://127.0.0.1:5000
```

---

## 🔑 Environment Variables

Create `.env` inside backend folder:

```env
GROQ_API_KEY=your_api_key_here
```
